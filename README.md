# EduBoard – Student Progress Dashboard

A clean, fully responsive student dashboard built with **pure HTML5 & CSS3** (Flexbox + Grid).

## Features
- Responsive sidebar & layout (mobile → desktop)
- Overview cards with GPA, attendance, credits
- Subject progress table with custom CSS progress bars
- Upcoming deadlines with color-coded badges
- Student profile card
- Semantic HTML & modern design
